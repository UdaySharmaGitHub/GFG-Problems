## About This Repository

This repository contains solutions for GFG problems organized by **difficulty level**. Each file is a self-contained C++ solution to a single problem. The goal is to:

- Build a reference library for common DSA patterns
- Track progress across difficulty levels
- Help others learn by reading clean, working solutions

## How to Contribute

Contributions are welcome! Follow these steps:

### 1. Fork & Clone

```bash
git clone https://github.com/<your-username>/GFG-Problems.git
cd GFG-Problems
```

### 2. Create a Branch

```bash
git checkout -b add/<Problem_Name>
```

### 3. Add Your Solution

- Place the file in the **correct difficulty folder** (`School Problems/`, `Basic Problems/`, `Easy Problems/`, `Medium Problems/`, or `Hard Problems/`)
- Follow the [naming convention](#naming-convention) below
- Make sure your file compiles without errors

### 4. Verify It Compiles

```bash
g++ -std=c++17 -Wall -o test "Easy Problems/Your_Problem_Name.cpp"
```

### 5. Commit and Push

```bash
git add "Easy Problems/Your_Problem_Name.cpp"
git commit -m "Add: Your_Problem_Name (Easy)"
git push origin add/<Problem_Name>
```

### 6. Open a Pull Request

Open a PR against the `main` branch. Describe the problem, your approach, and the time/space complexity.

## Naming Convention

File names follow this pattern:

```
Problem_Name_With_Underscores.cpp
```

Rules:
- Use **underscores** (`_`) instead of spaces
- Use **Title_Case** for each word
- Match the **exact problem name** from GFG as closely as possible
- `.cpp` extension only — do not commit `.exe` or compiled binaries

**Examples:**
```
✅  Kadane's_Algorithm.cpp
✅  Lowest_Common_Ancestor_in_a_BST.cpp
✅  Reverse_a_linked_list.cpp
❌  kadanes algorithm.cpp
❌  solution1.cpp
❌  Kadane's_Algorithm.exe
```
